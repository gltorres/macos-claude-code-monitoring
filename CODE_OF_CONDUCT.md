# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][covenant].

In short: be respectful, assume good intent, and keep discussion focused on
the work. Harassment, discrimination, or personal attacks are not acceptable
in issues, PRs, commits, or any other project space.

## Reporting

Open a private report via the repo's
[GitHub Security Advisories](https://github.com/OWNER/REPO/security/advisories/new)
tab. Reports are handled privately; we aim to acknowledge within 7 days.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
